# SchoolGPT Reporting Functions

Standalone Python Azure Functions project for safeguarding-led reporting.

Phase 1 scope:
- daily DSL incident report
- daily aggregate usage summary
- DSL keyword-watch report
- safe manual operational run path

Manual run examples:
- `python manual_run.py dsl-daily`
- `python manual_run.py usage-daily`
- `python manual_run.py keyword-watch`
